# Getting started
1. Enable annotation processing
2. Run application with file Application.java

    2.1 REST application
    
    [Swagger](http://localhost:8080/swagger-ui/index.html?configUrl=/v3/api-docs/swagger-config)
    
    2.2 CONSOLE application
             
    Use VM option to start application in console mode:         
    
             -Dspring.profiles.active=console
